# Small Office Network Simulation

## Project Overview
This project demonstrates a small office network topology using Cisco Packet Tracer.

## Devices Used
- Cisco 1941 Router
- 2 x Cisco 2960 Switches
- 4 PCs

## Configuration
- Assigned static IP addresses
- Configured default gateways
- Enabled inter-network communication
- Verified connectivity using ICMP ping

## Outcome
Successful communication between all PCs across different networks.
